Cloud Deployment¶

A PostgreSQL server can be deployed on the Amazon AWS and EDB BigAnimal cloud using this module. In future more cloud options will be available.

To launch the Cloud Deployment… tool, right click on the Server Group or Server of the tree control, and select Deploy a Cloud Instance from the Register menu.

  • Amazon AWS RDS Cloud Deployment
  • EDB BigAnimal Cloud Deployment
  • Azure PostgreSQL Cloud Deployment
